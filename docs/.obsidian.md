# ⚡ Obsidian Vault — Job Finder

Este directorio es un **vault de Obsidian**. Abrilo directamente en Obsidian:

1. Abrí Obsidian
2. "Open folder as vault" → seleccioná `D:\work-scrapper\job-finder`
3. Listo ✓

## Plugins recomendados
- **Obsidian Git** — para versionar notas
- **Dataview** — para consultar jobs desde las notas
- **Templater** — para templates de daily digest
